Donar format a text amb Markdown¶

Resum de les regles bàsiques:¶

El format es defineix amb etiquetes especfiques per a cada cas:

  • El parragraf es forma amb dos return.
  • El salt de lnia amb dos espais seguits.
  • Les cites amb > al principi del text
  • Text en itlica : *...* o _..._
  • Text en negreta : **...** o __...__
  • Ttols : #
  • Llistes desordenades : * + espai o _ + espai o + + espai. Per als subnivells es pot precedir d'un espai o el tabulador.
  • Llistes ordenades : 1., 2., etc.
  • Codi, forma 1 : entre dos `` (``codi``). Equival al \<code> de l'html.
  • Codi, forma 2 : precedit de quatre espais en cada lnia. Equival al \<pre> de l'html.
  • Blocs de codi : entre dues lnies formades per tres ~ (~~~)
  • Enllaos ocultant l'adrea : el nom entre [] i l'adrea de l'enlla entre ()
  • Enllaos mostrant l'adrea : adrea continguda entre \< i >

  • Imatges : !text alternatiu

  • *** sense text : Dibuixa una lnia horitzontal

Donar estil al text¶

  • Itlica : contingut entre asteriscs o subguions : *text* o _text_
    • Estic entre subguions. Altre exemple: sc un text normal i jo en itlica.
  • Negreta : contingut entre dobles asteriscs o subguions: **text** o __text__
    • Estic entre dobles subguions. Jo sc un text normal i jo estic entre dobles asteriscs.

Títols¶

  • Els ttols van precedits del smbol #.
  • N'hi ha fins a sis nivells, en progressi descendent (equivalents a l'etiqueta h1, h2, etc., de l'html).
    • Exemple: # Aix ser un ttol de nivell 1
  • Es poden 'escapar' amb la contrabarra \ :
